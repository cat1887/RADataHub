Reasons for missing data

If data is missing in the dashboard the reason will be displayed in the data table. The reasons are:

    • [a] Data not available (when the data is unavailable, for example because a police force was not able to provide data for a given period)
    • [b] Suppressed due to low volumes (only applicable to ' Witnesses who have given evidence via Section 28')
    • [c] No data to calculate this field (when it is not possible to calculate a metric, for example timeliness metrics when there are zero offences/cases)

Data rounding

All charts plot unrounded data but the numbers in the table or when you hover over the chart are rounded. You can see the unrounded numbers in the CSV file.
